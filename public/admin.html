Admin Panel view

Router panels

Each Cerebrum panel is identified by its fixed IP. Assign which heads it controls, its screen layout, and arrange them to match the physical monitor wall.

Head snapshot filters (global)

Set which snapshots are permitted for each head. These apply to that head on every panel. A panel can override a head with “All snapshots” to show everything regardless of this filter.

Auto-share snapshots

Continuously checks the selected cards and marks any unshared snapshot as shared, so new snapshots propagate across boards automatically.

Scheduled backup

Exports a target board's snapshots on a daily schedule, dated and retained for a set number of days. Files download from here — nothing is left on the Neuron.
